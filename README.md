# India CPI Inflation Analysis Dashboard (2021–2025)

## Project Overview

This project analyzes India's Consumer Price Index (CPI) inflation trends from 2021–2025 using Excel and Power BI.

The dashboard focuses on:

* Inflation trend analysis (YoY%)
* Monthly inflation change (MoM%)
* Inflation volatility
* Key inflation indicators such as average, highest, and lowest inflation rates

The objective of this project is to demonstrate data analysis, economic interpretation, and dashboard visualization skills using real macroeconomic indicators.

## Key Insights

* India experienced sustained inflationary pressure during 2022–2023, with CPI inflation remaining consistently elevated.
* Peak inflation reached approximately **7.79%**, reflecting significant price pressures during the period.
* Inflation showed signs of moderation during 2024–2025, indicating easing inflationary trends.
* Current inflation declined to around **1.55%**, suggesting reduced price pressures in the economy.
* Monthly inflation changes highlighted periods of volatility and disinflation across different phases.
